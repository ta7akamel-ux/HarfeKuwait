# موقع حرفي الكويت للألومنيوم - MVP

## وصف المشروع
موقع ويب تجريبي (MVP) احترافي لحرفي ألومنيوم متخصص في صيانة وتصميم مطابخ الألومنيوم بالكويت. الموقع مصمم باللهجة الكويتية البحتة مع تحسين شامل لمحركات البحث.

## كيفية تشغيل الموقع

### 1. تشغيل محلي
```bash
cd aluminum-kitchen-mvp
python3 -m http.server 8000
```
ثم افتح المتصفح على: `http://localhost:8000`

### 2. رفع على استضافة
- ارفع جميع الملفات إلى خادم الويب
- تأكد من أن ملف index.html في المجلد الجذر
- حدث الروابط في sitemap.xml و robots.txt
