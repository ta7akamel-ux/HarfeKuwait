# دليل إعداد EmailJS لربط النماذج بالإيميل

## الخطوة 1: إنشاء حساب EmailJS

1. اذهب إلى [EmailJS](https://www.emailjs.com/)
2. اضغط "Sign Up" وأنشئ حساب جديد
3. أكد بريدك الإلكتروني

## الخطوة 2: إعداد خدمة البريد الإلكتروني

1. في لوحة التحكم، اضغط "Email Services"
2. اضغط "Add New Service"
3. اختر "Gmail" (أو أي خدمة بريد أخرى)
4. أدخل بيانات `harfekuwait@gmail.com`
5. احفظ الـ Service ID

## الخطوة 3: إنشاء قوالب الرسائل

### قالب رسائل التواصل:
1. اذهب إلى "Email Templates"
2. اضغط "Create New Template"
3. استخدم هذا القالب:

```
الموضوع: رسالة جديدة من موقع حرفي الكويت للألومنيوم

مرحباً،

تم استلام رسالة جديدة من الموقع:

الاسم: {{from_name}}
البريد الإلكتروني: {{from_email}}
رقم الهاتف: {{phone}}

الرسالة:
{{message}}

---
تم الإرسال من موقع حرفي الكويت للألومنيوم
```

### قالب التقييمات:
1. أنشئ قالب جديد للتقييمات:

```
الموضوع: تقييم جديد من عميل - {{stars}}

مرحباً،

تم استلام تقييم جديد:

اسم العميل: {{customer_name}}
التقييم: {{stars}} ({{rating}}/5)

التعليق:
{{feedback}}

---
تم الإرسال من موقع حرفي الكويت للألومنيوم
```

## الخطوة 4: تحديث الكود

1. افتح ملف `js/script.js`
2. استبدل هذه القيم:

```javascript
// في بداية الملف
emailjs.init("YOUR_PUBLIC_KEY"); // ضع المفتاح العام هنا

// في دالة sendContactEmail
await emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', {
    // ضع معرف الخدمة ومعرف القالب
});

// في دالة sendRatingEmail
await emailjs.send('YOUR_SERVICE_ID', 'YOUR_RATING_TEMPLATE_ID', {
    // ضع معرف الخدمة ومعرف قالب التقييم
});
```

## الخطوة 5: الحصول على المعرفات

1. **Public Key**: من "Account" > "General"
2. **Service ID**: من "Email Services" > اختر الخدمة
3. **Template ID**: من "Email Templates" > اختر القالب

## الخطوة 6: اختبار النظام

1. ارفع الملفات المحدثة على Netlify
2. جرب إرسال رسالة من الموقع
3. تأكد من وصول الرسالة إلى `harfekuwait@gmail.com`

## ملاحظات مهمة:

- EmailJS مجاني لـ 200 رسالة شهرياً
- يمكن ترقية الحساب للمزيد من الرسائل
- إذا لم يعمل EmailJS، سيفتح النظام تطبيق البريد الافتراضي
- تأكد من تفعيل "Less secure app access" في Gmail (إذا لزم الأمر)

## بديل: استخدام Netlify Forms

إذا كنت تفضل استخدام Netlify Forms بدلاً من EmailJS:

1. أضف `netlify` إلى كل form:
```html
<form netlify name="contact">
```

2. سيتم إرسال الرسائل إلى لوحة تحكم Netlify
3. يمكن إعداد إشعارات بريد إلكتروني من Netlify

## الدعم الفني

إذا واجهت أي مشاكل في الإعداد، يمكنك:
1. مراجعة وثائق EmailJS
2. التواصل مع الدعم الفني
3. استخدام البديل المدمج (فتح تطبيق البريد)
